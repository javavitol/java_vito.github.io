docker容器的数据卷

2023-01-28 docker

数据卷概念及作用
配置数据卷
配置数据卷容器

数据卷概念（解决以下问题）：
Docker容器删除后，在容器中产生的数据不存在
Docker容器和外部机器不可以直接交换文件
容器之间进行数据交互

数据卷

数据卷是宿主机中一个目录或文件
当容器目录和数据卷目录绑定后，对方的修改会立即同步
一个数据卷可以被多个容器同时挂载
一个容器也可以被挂载多个数据卷

配置数据卷

创建启动容器时，使用 -v 参数， 设置数据卷

1.目录必须是绝对路径
2.如果目录不存在，会自动创建
3.可以挂载多个数据卷

配置数据卷容器

1.创建数据卷容器， 使用-v参数设置数据卷
2./volume 是容器文件，宿主未设置目录，会自动分配

创建启动 C1,C2容器，使用 –volumes-from参数，设置容器的数据卷
